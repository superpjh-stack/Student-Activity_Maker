## 1. Overview

### 1.1 Purpose

현재 서비스는 8개 과목에 대해 범용 주제 풀(pool)을 제공하지만, 학년별 교육과정 수준이 반영되지 않아
고2 학생의 심화 탐구 요구와 대입 학생부종합전형(학종) 차별화 요구를 충족하지 못한다.
이 플랜은 고2 수준에 맞춘 전용 주제 뱅크 50개를 정의하고 이를 서비스에 반영하는 방법을 규정한다.

### 1.2 Background

- 고2는 문이과 분화가 시작되며 심화 수학·과학 교육과정이 적용된다.
- 학종 입학사정관은 "누가 써도 같은 보고서"를 즉시 걸러낸다. 주제 자체의 차별성이 관문이다.
- 현재 `src/lib/topics-cache.ts` 주제 풀은 학년 구분 없이 단일 풀로 운영된다.
- 고2 전용 주제를 추가하면 서비스의 타깃 집중도와 사용자 만족도가 함께 올라간다.

### 수학 (고2)

**1학기 — 탐구 설계 중심**

1. **로지스틱 성장 모델로 분석하는 감염병 확산 수학** — 지수함수와 시그모이드 곡선을 활용해 코로나19 실제 확진 데이터를 피팅하고 R0 값을 도출하는 탐구 | 진로: 수리통계학, 보건정책학, 의대
2. **몬테카를로 시뮬레이션으로 구하는 원주율과 불확실성** — 난수 생성과 기댓값 이론으로 원주율을 추정하고, 표본 수에 따른 수렴 속도를 분석하는 확률 탐구 | 진로: 통계학, 데이터사이언스, 금융공학
3. **베지에 곡선의 수학: 애니메이션 속 미분기하학** — 3차 베지에 곡선의 매개변수 방정식과 드 카스텔조 알고리즘을 탐구하고 픽사·디즈니 애니메이션 실제 적용 사례 분석 | 진로: 컴퓨터그래픽스, 소프트웨어공학
4. **최적화 문제로 본 물류 네트워크 — 선형계획법의 실전** — 편의점 배송 경로를 선형계획법과 심플렉스법으로 최적화하고 실제 물류비 절감 효과를 수치로 비교 | 진로: 산업공학, 경영과학, 물류학
5. **푸리에 급수로 분해하는 음악의 수학** — 악기 소리의 파형 데이터를 수집해 퓨리에 급수로 분해하고 배음 구조와 음색의 관계를 수학적으로 규명 | 진로: 음향공학, 신호처리, 수학교육

**2학기 — 비판적 분석 및 대안 제시 중심**

1. **AI 알고리즘의 수학적 편향 — 행렬 분해와 추천 시스템 비판** — 협업 필터링에 사용되는 SVD(특이값 분해) 행렬 수학을 분석하고 추천 시스템이 특정 집단에 편향되는 수학적 원인을 비판적으로 탐구 | 진로: AI/ML공학, 데이터윤리, 통계학
2. **기후 모델의 수학 — 미분방정식으로 본 지구온난화 예측의 불확실성** — 로렌츠 방정식과 카오스 이론을 적용해 기후 예측 모델의 민감도를 분석하고, 불확실성 정량화 방법을 비판적으로 평가 | 진로: 기상학, 환경공학, 응용수학
3. **암호화폐 가격은 랜덤워크인가 — 주식/코인 데이터로 검정하는 효율적 시장 가설** — 실제 비트코인 일별 수익률 데이터로 정규성 검정, 자기상관 분석을 수행해 효율적 시장 가설의 수학적 타당성 검증 | 진로: 금융공학, 계량경제학, 수학
4. **소수(Prime) 분포의 비밀 — 리만 가설과 소수정리의 시각화** — 소수계량 함수 π(x)와 자연로그 근사 오차를 직접 계산하고 리만 제타 함수의 trivial zeros를 시각화해 현대 수학의 미해결 문제를 탐구 | 진로: 순수수학, 정수론, 암호학
5. **투표 시스템의 수학적 불가능성 — 애로의 불가능성 정리 검증** — 다수결·보다점수제·콩도르세 방식 투표 결과를 수학적으로 비교해 애로 정리(Arrow's impossibility theorem)의 현실 적용 사례를 분석하고 대안 제시 | 진로: 사회선택이론, 정치학, 경제수학

### 물리 (고2)

**1학기 — 탐구 설계 중심**

1. **초전도체의 마이스너 효과 측정 — 임계 온도와 자기장 차폐 실험** — YBCO 고온 초전도체를 액체질소(77K)로 냉각해 부상 높이와 자기장 세기 관계를 측정하고 BCS 이론과 비교 분석 | 진로: 응집물질물리학, 재료공학, 전기공학
2. **스마트폰 가속도계로 구현하는 단순조화운동 — 용수철 상수 정밀 측정** — 스마트폰 내장 가속도 센서와 PhyPhox 앱으로 용수철-질량 시스템의 SHM 데이터를 수집하고 이론값과 오차 분석 | 진로: 물리학, 계측공학, 스마트센서
3. **빛의 편광과 말뤼스 법칙 — 편광판 각도에 따른 광 세기 정밀 측정** — 편광판 2장과 광센서로 말뤼스 법칙(I = I₀cos²θ)을 검증하고 LCD 디스플레이 편광 원리와 연결해 분석 | 진로: 광학공학, 디스플레이공학, 물리학
4. **유체 점성 측정 — 스토크스 법칙으로 결정하는 글리세린 점도** — 구슬 낙하 종단속도로 스토크스 법칙을 적용해 글리세린 점도를 측정하고 온도 변화에 따른 점도 변화 데이터를 분석 | 진로: 유체역학, 화학공학, 바이오의공학
5. **광전 효과의 재현 — LED 역전압법으로 플랑크 상수 결정** — 서로 다른 파장의 LED에 역전압을 가해 정지 전위를 측정하고 플랑크 상수를 직접 결정해 양자역학의 역사적 실험을 재현 | 진로: 양자물리학, 반도체공학, 물리학

**2학기 — 비판적 분석 및 대안 제시 중심**

1. **핵융합 에너지의 물리 — ITER 플라즈마 가둠의 원리와 한계 비판** — 토카막 자기 가둠의 물리 원리(로런츠 힘, 플라즈마 불안정성)를 분석하고, 현재 Q값(에너지 이득)이 1을 넘지 못하는 물리적 한계를 비판적으로 탐구 | 진로: 핵융합공학, 플라즈마물리학, 에너지정책
2. **중력파 검출의 물리 — LIGO 마이켈슨 간섭계 원리 재구성** — 마이켈슨 간섭계로 빛의 경로 차이를 측정하는 실험을 설계하고, LIGO가 양성자 크기의 1/1000을 측정하는 물리적 메커니즘을 분석 | 진로: 천체물리학, 정밀계측, 중력물리학
3. **전기차 배터리 열폭주 — 리튬이온 셀의 열물리학 분석** — 리튬이온 배터리의 열폭주(thermal runaway) 체인반응 메커니즘을 열역학 제1·2법칙으로 분석하고 BMS(배터리관리시스템)의 물리적 대응 원리를 비판적으로 평가 | 진로: 에너지공학, 전기차공학, 배터리재료학
4. **음속 측정과 대기 상태 — 온도·습도 변수 통제 실험** — 기온·습도를 변화시키며 공명관법으로 음속을 측정하고 Newton-Laplace 공식과 비교해 이상기체 가정의 한계를 실험적으로 규명 | 진로: 음향공학, 기상학, 응용물리학
5. **태양광 패널 효율의 물리적 한계 — 쇼클리-케이서 한계와 현실 데이터 비교** — 단일 p-n 접합 태양전지의 이론 최대 효율(Shockley-Queisser limit, ~33%)을 도출하고 실제 시판 패널 효율 데이터와 비교 분석 | 진로: 태양광공학, 반도체물리학, 신재생에너지

### 화학 (고2)

**1학기 — 탐구 설계 중심**

1. **카페인 추출 효율 비교 — 용매 극성과 분배 계수 실험** — 물/에틸아세테이트/DCM 용매별 카페인 추출 수율을 TLC로 비교하고 분배 계수(Kd)와 용매 극성 지수 상관관계 분석 | 진로: 유기화학, 제약화학, 식품공학
2. **광촉매 산화티타늄의 메틸렌블루 분해 반응 속도론** — TiO₂ 광촉매량과 UV 조사 시간을 변수로 메틸렌블루 흡광도 변화를 측정하고 유사 1차 반응 속도 상수(k) 결정 | 진로: 환경화학, 촉매공학, 화학공학
3. **완충 용액의 완충 능력 정량 측정 — Henderson-Hasselbalch 방정식 검증** — 아세트산/아세트산나트륨 완충계를 만들고 강산·강염기 첨가량에 따른 pH 변화를 실험적으로 측정해 이론값과 비교 | 진로: 생화학, 의약학, 화학교육
4. **전기화학적 구리 전착 — 전류 밀도와 석출 질량 패러데이 법칙 검증** — 황산구리 수용액 전기분해에서 전류·시간·전극 면적을 변수로 조절해 패러데이 법칙 비례 상수를 직접 결정 | 진로: 전기화학, 반도체도금, 재료공학
5. **향수 분자의 휘발성 — 증기압과 끓는점 관계 실험** — 서로 다른 향료 성분(리날로올, 시트로넬롤 등)의 증기압을 측정하고 클라우지우스-클라페이론 방정식으로 증발엔탈피를 결정 | 진로: 향료화학, 화장품과학, 물리화학

**2학기 — 비판적 분석 및 대안 제시 중심**

1. **미세플라스틱의 중금속 농집 — 환경 화학 위기 비판적 분석** — 폴리스티렌 미세플라스틱과 납·카드뮴 이온 흡착 실험을 설계하고, 해양 미세플라스틱이 먹이사슬을 통해 인체로 유입되는 화학적 경로를 비판적으로 분석 | 진로: 환경화학, 독성학, 해양환경학
2. **리튬이온 배터리 전해질 — 고체 전해질 vs 액체 전해질의 화학적 비교** — 기존 유기 전해질(EC/DMC)과 황화물계 고체 전해질의 이온 전도도와 전기화학적 안정 범위를 비교해 전고체 배터리 상용화의 화학적 장벽 분석 | 진로: 배터리화학, 재료화학, 전기화학
3. **가정용 세제의 계면활성제 농도와 표면장력 — CMC 결정 실험** — 도데실황산나트륨(SDS) 농도를 변화시키며 Wilhelmy 판법으로 표면장력을 측정하고 임계 미셸 농도(CMC)를 결정해 환경친화적 세제 설계에 적용 | 진로: 화장품화학, 계면화학, 환경공학
4. **대기 오염 물질 NOx의 생성 화학 — 자동차 연소 온도와 NOx 상관관계** — 연소 온도에 따른 제알도비치 메커니즘(Zeldovich mechanism) 반응 속도를 분석하고, 촉매 변환기의 SCR 탈질 화학 반응 효율을 비판적으로 평가 | 진로: 대기화학, 자동차공학, 환경화학
5. **의약품 속 불순물 — 아스피린 합성과 철(III) 이온 순도 검사** — 살리실산과 무수아세트산으로 아스피린을 직접 합성하고 FeCl₃ 정색 반응과 재결정으로 순도를 검증해 의약품 품질관리 GMP 기준과 비교 | 진로: 제약학, 유기화학, 의약화학

### 생물 (고2)

**1학기 — 탐구 설계 중심**

1. **CRISPR-Cas9 원리 시뮬레이션 — sgRNA 설계와 표적 특이성 분석** — 실제 gRNA 설계 프로그램(Benchling)으로 특정 유전자의 sgRNA를 설계하고 off-target 예측 점수를 분석해 유전자 가위 정밀도의 한계를 탐구 | 진로: 분자생물학, 유전공학, 의학
2. **효모 발효 최적 조건 탐구 — 온도·pH·당 농도 변수 설계 실험** — 사카로미세스 세레비시아에 발효에서 CO₂ 발생량을 지표로 최적 온도(25~40°C), pH(4~7), 포도당 농도(5~20%) 조건을 3요인 실험 설계로 탐구 | 진로: 발효공학, 식품생물학, 생명공학
3. **광합성 속도의 광보상점 결정 — 수생식물 기포 계수법** — Elodea(물풀)를 활용해 광도 변화에 따른 기포 발생률을 측정하고 광보상점·광포화점을 결정해 C3/C4 식물 비교 탐구로 확장 | 진로: 식물생리학, 농업과학, 생태학
4. **피부 미생물군(마이크로바이옴) — 손 세척 전후 세균 배양 비교** — 손 세척 전·후, 세정제 종류별 세균 군집을 한천 배지에 배양하고 콜로니 형성 단위(CFU)를 정량 비교해 항균 효능 분석 | 진로: 미생물학, 공중보건학, 의학
5. **세포 삼투 현상 정량화 — 감자 디스크 질량 변화와 삼투 퍼텐셜 계산** — 감자 디스크를 0~1.0M 수크로스 용액에 침지해 질량 변화를 측정하고 등장 농도와 세포 삼투 퍼텐셜(Ψs)을 계산 | 진로: 세포생물학, 식물학, 의생명과학

**2학기 — 비판적 분석 및 대안 제시 중심**

1. **항생제 내성균 출현 메커니즘 — 진화유전학 관점 비판적 분석** — 대장균 항생제 내성 획득의 돌연변이·수평 유전자 이동 메커니즘을 분석하고, 항생제 과처방 데이터와 내성균 출현율 상관관계를 비판적으로 평가 | 진로: 미생물학, 감염병학, 의학
2. **장내 미생물과 뇌-장 축 — 정신건강과 장내균총의 관계 분석** — 장-뇌 신경축(gut-brain axis)의 미주신경·세로토닌 경로를 분석하고 임상 데이터로 장내 미생물 다양성과 우울증 발병률 상관관계를 탐구 | 진로: 신경과학, 정신의학, 장내미생물학
3. **GMO 콩의 제초제 내성 유전자 — 글리포세이트 저항성 메커니즘과 생태계 영향 비판** — EPSPS 유전자 돌연변이로 인한 글리포세이트 내성 메커니즘을 분석하고 제초제 내성 잡초 확산 데이터로 GMO 생태계 영향을 비판적으로 평가 | 진로: 분자유전학, 농업생물공학, 환경생태학
4. **노화의 텔로미어 이론 검증 — 세포 분열 횟수와 텔로미어 단축 데이터 분석** — 헤이플릭 한계와 텔로미어 단축 이론을 문헌 데이터로 검증하고, 암세포의 텔로머라제 과발현이 불사 형질을 부여하는 메커니즘을 분석 | 진로: 노화의학, 종양학, 세포생물학
5. **뇌 가소성과 학습 — 장기강화(LTP)의 분자 메커니즘과 교육적 시사점** — AMPA/NMDA 수용체 기반 LTP 메커니즘을 분석하고, 인지 과학 연구 데이터를 활용해 최적 학습 간격(간격 효과)의 신경생물학적 근거를 도출 | 진로: 신경과학, 인지과학, 의학
