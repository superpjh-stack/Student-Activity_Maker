## 1. Overview

### 1.1 Purpose

현재 서비스는 8개 과목 각 10개 주제를 제공하지만, 학년별 커리큘럼 연계성이 부족하다. 고1 학생은 처음 세특을 접하는 단계로, "이 주제가 내 수준에 맞나?", "어떤 탐구를 해야 생기부에 임팩트가 있나?"를 모른다. 고1 전용 큐레이션 주제 은행을 도입하여 학년·학기 맞춤 탐구 주제를 제공한다.

### 1.2 Background

- 고1은 교육과정상 기초 개념 학습 단계 (수학: 다항식/방정식, 물리: 역학 입문, 화학: 물질의 구성, 생물: 세포·유전 기초, 생윤: 윤리 이론)
- AI 트렌드(ChatGPT, 기후위기, 바이오테크) 반영 주제가 입학사정관에게 임팩트를 줌
- 1학기(기초 탐구)와 2학기(응용·융합 탐구)의 난이도 및 방법론을 구분해야 세특 스토리가 자연스럽게 쌓임

### 수학 (고1)

**1학기 — 기초 탐구 (다항식, 방정식, 부등식 범위)**

1. **황금비와 자연·건축의 수학적 패턴** — 피보나치 수열과 황금비(φ≈1.618)가 해바라기 씨앗 배열·파르테논 신전 비율에 나타나는 이유를 수식으로 분석 | 진로: 건축학과·수학교육학과
   - 탐구 방법: 문헌조사 + 실측(자 사용) + 비율 계산
   - 임팩트 이유: 추상적 수열을 실생활로 연결해 수학적 감수성을 보여줌

2. **이차방정식으로 해석하는 공 던지기 포물선** — 포물선 운동 데이터를 스마트폰 카메라로 포착해 y = ax²+bx+c 모형 피팅 | 진로: 물리학과·스포츠공학
   - 탐구 방법: 실험(공 던지기 동영상) + 좌표 추출 + 회귀 피팅

3. **불평등 분포와 부등식 — 지니계수의 수학** — 로렌츠 곡선·지니계수를 부등식과 넓이 개념으로 설명 | 진로: 경제학과·사회학과
   - 탐구 방법: 공공 통계 데이터 분석 (OECD stat)

4. **다항식 전개와 암호화 원리 — RSA 맛보기** — 다항식·인수분해 원리를 활용해 공개키 암호의 핵심 개념(소인수 분해의 어려움)을 고1 수준으로 설명 | 진로: 컴퓨터공학·정보보안학과
   - 탐구 방법: 수학적 증명 + 간단한 코딩(Python/스크래치)

5. **스마트폰 화면 해상도와 비율의 수학** — 화면 비율(16:9, 21:9)과 PPI 계산에 적용되는 피타고라스·비율 개념 탐구 | 진로: 전자공학·UX디자인
   - 탐구 방법: 실측 + 계산 + 제조사 스펙 비교

**2학기 — 응용·융합 탐구 (함수, 수열, 확률·통계 범위)**

6. **기후 데이터로 배우는 일차함수와 선형 회귀** — 최근 10년 서울 평균기온 데이터로 선형 추세를 분석하고, 기울기가 의미하는 온난화 속도를 해석 | 진로: 환경공학·기상학
   - 탐구 방법: 데이터 분석(기상청 공공 API) + 그래프 해석

7. **로그함수로 이해하는 AI 학습의 손실 함수** — 머신러닝의 Cross-Entropy Loss가 왜 로그를 사용하는지 수학적으로 탐구 | 진로: AI공학·컴퓨터공학
   - 탐구 방법: 수학적 유도 + Python 시뮬레이션

8. **등비수열과 복리의 힘 — 노후 자산 시뮬레이션** — 등비수열 합 공식으로 20년 복리 투자 시나리오를 계산하고 인플레이션 변수 추가 | 진로: 경영학·금융공학
   - 탐구 방법: 스프레드시트 시뮬레이션 + 수식 유도

9. **확률로 분석하는 유튜브 알고리즘의 추천 원리** — 조건부 확률과 베이즈 정리를 활용해 영상 추천 알고리즘의 수학적 구조를 설명 | 진로: 데이터과학·컴퓨터공학
   - 탐구 방법: 문헌조사 + 수식 정리 + 사례 검증

10. **통계로 보는 입시 공정성 — 표준점수와 백분위의 함정** — 정규분포, 표준편차, Z-점수 개념으로 수능 표준점수 산출 방식을 비판적으로 분석 | 진로: 통계학·교육학·사회학
    - 탐구 방법: 수능 데이터 분석 + 통계 계산

### 물리 (고1)

> 과학 과목 — 모든 탐구보고서에 실험 측정값·단위·오차 범위 포함 필수 (CLAUDE.md Rule 2)

**1학기 — 기초 탐구 (역학, 힘과 운동)**

1. **스마트폰 가속도 센서로 측정하는 중력가속도(g)** — 낙하 실험에서 앱(phyphox)으로 g를 측정하고 이론값 9.8 m/s²과 오차 분석 | 진로: 물리학과·항공우주공학
   - 실험 데이터: 낙하 거리 3종(50cm/100cm/150cm) × 5회 반복, 평균±표준편차
   - 임팩트 이유: 디지털 기기로 직접 측정·오차 처리 능력을 보여줌

2. **마찰력의 두 얼굴 — 정지마찰과 운동마찰 비교 실험** — 재질(나무·유리·고무)별 마찰계수를 측정해 미끄럼 방지 타일 설계에 적용 | 진로: 재료공학·건축공학
   - 실험 데이터: 재질 3종 × 하중 3단계, 최대 정지마찰력 측정

3. **자전거 기어와 지레의 원리 — 일과 힘의 상관관계** — 기어비 변화에 따른 페달 힘과 이동 거리를 측정해 에너지 효율 계산 | 진로: 기계공학·스포츠과학
   - 탐구 방법: 실측 + 기어비 계산 + 에너지 보존 법칙 적용

4. **소리의 속도 측정과 도플러 효과의 실생활** — 메아리 시간 측정 또는 앱(Spectroid)으로 구급차 사이렌 주파수 변화 분석 | 진로: 음향공학·의공학
   - 실험 데이터: 거리별(10m/20m/30m) 소리 왕복 시간 측정

5. **투석기 설계로 배우는 포물선 운동 최적화** — 발사각도별 최대 수평 거리를 실험·이론값과 비교해 45도 최적 발사각 검증 | 진로: 기계공학·물리교육
   - 실험 데이터: 발사각 5종(30°/35°/40°/45°/50°) × 3회 측정

**2학기 — 응용·융합 탐구 (전기·자기, 파동, 에너지)**

6. **태양광 패널 효율 측정과 에너지 전환율 분석** — 각도·조도별 태양광 패널 전력 출력을 측정해 최적 설치각도 도출 | 진로: 에너지공학·환경공학
   - 실험 데이터: 각도 7종(0°~90°, 15° 간격) × 조도 2조건

7. **전자기 유도로 만드는 미니 발전기 — 코일 수의 영향** — 코일 감는 수(10/20/30/40회)에 따른 유도 기전력 측정 및 비선형 관계 분석 | 진로: 전기공학·에너지공학

8. **파동의 간섭과 소음 제거 이어폰의 원리** — 두 스피커 파동의 보강/소멸 간섭을 시각화하고 ANC(능동 소음 제거) 원리와 연계 | 진로: 음향공학·전자공학

9. **적외선 온도계와 열복사 법칙 — 물체 온도와 방출 에너지** — 색이 다른 표면(흑색·백색·은박지)의 열복사 차이를 적외선 온도계로 측정 | 진로: 재료공학·건축환경

10. **지진파 분석과 P파·S파의 속도 비교** — 지진 데이터(KMA 공공 데이터)로 P파·S파 도달 시간 차이를 분석해 진앙 거리 추정 | 진로: 지구과학·방재공학

### 화학 (고1)

> 과학 과목 — 실험 측정값·농도·온도 조건 포함 필수

**1학기 — 기초 탐구 (물질의 구성, 원자·분자, 화학 결합)**

1. **음료수 속 산·염기 — 적정 실험으로 측정하는 구연산 농도** — 콜라·오렌지주스에 NaOH로 산염기 적정 실험, 구연산 농도(mol/L) 정량 분석 | 진로: 식품공학·약학
   - 실험 데이터: 샘플 3종 × 3회 적정, 종말점 부피(mL)±0.05mL

2. **주기율표 원소 분류와 불꽃반응 색의 과학** — Li·Na·K·Ca·Sr 불꽃 반응으로 발광 파장과 전자 에너지 준위 관계 탐구 | 진로: 화학과·분석화학
   - 실험 데이터: 각 원소 RGB 색상값 + 참조 파장(nm) 비교

3. **비누 만들기와 계면활성제의 화학 — 에스터화 반응** — 올리브유·수산화나트륨으로 비누 제조 후 세정력(기름 제거율) 정량 측정 | 진로: 화학공학·재료공학
   - 실험 데이터: 기름 초기 질량 대비 제거율(%) 3회 측정

4. **양배추 지시약으로 배우는 pH 척도 — 천연 색소의 화학** — 안토시아닌(적색 양배추 추출물)으로 pH 1~14 색 변화 차트 작성 | 진로: 화학교육·식품공학

5. **화학 결합의 종류와 물질 성질 — 이온·공유·금속 결합 비교** — 소금·설탕·알루미늄의 전기 전도도·용해도·녹는점을 체계적으로 측정해 결합 유형과 물성 관계 귀납 | 진로: 재료공학·화학공학

**2학기 — 응용·융합 탐구 (화학 반응, 산화·환원, 생명화학)**

6. **철의 부식 속도 조절 — 환경 변수(온도·pH·염도) 실험** — 철판 표면적 변화량을 무게 감소로 측정, 부식 억제제(베이킹소다) 효과 비교 | 진로: 재료공학·부식방지공학
   - 실험 데이터: 온도 3조건(15°C/25°C/35°C) × 시간 48h, 무게 감소량(g)

7. **광합성 색소 크로마토그래피와 엽록소 분리** — 시금치·붉은 단풍잎 색소를 TLC(종이 크로마토그래피)로 분리해 Rf값 비교 | 진로: 생화학·식물공학

8. **플라스틱 분해 조건 탐구 — 자외선과 미생물의 역할** — UV 노출·토양 매립 조건별 PLA 바이오 플라스틱 분해 속도를 무게 변화로 측정 | 진로: 환경공학·생명공학
   - 실험 데이터: 3조건(UV/토양/대조군) × 4주간 주 1회 무게 측정

9. **전기분해로 만드는 수소연료 — 전력과 기체 발생량의 관계** — 물 전기분해에서 전류 세기(mA)별 수소 발생량(mL/min) 측정, 패러데이 법칙 검증 | 진로: 에너지공학·수소경제

10. **아스피린 합성과 순도 검증 — 의약품 화학의 기초** — 살리실산+무수아세트산으로 아스피린 합성 후 재결정·적정으로 순도(%) 계산 | 진로: 약학·화학공학

### 생물 (고1)

> 과학 과목 — 실험 측정값·대조군·실험군 비교 포함 필수

**1학기 — 기초 탐구 (세포, 유전, 생명의 특성)**

1. **양파 세포 삼투압 실험 — 농도에 따른 세포 크기 변화** — 0%·0.5%·1%·2%·3% NaCl 용액에서 양파 세포 크기 현미경 측정, 등장·고장·저장 비교 | 진로: 의학·생명과학
   - 실험 데이터: 농도 5조건 × 세포 20개 측정, 평균 세포 지름(μm)±SD

2. **혈액형 유전 — ABO 혈액형 유전 패턴 시뮬레이션** — 부모 유전자형 조합별 자녀 혈액형 확률을 표로 작성하고 멘델 법칙과 연결 | 진로: 유전상담학·의학
   - 탐구 방법: 유전 퍼넷 표 작성 + 가상 족보 시뮬레이션

3. **효소 반응과 온도 — 카탈레이스 활성 측정** — 감자 추출물(카탈레이스)이 H₂O₂를 분해할 때 산소 기포 발생 속도를 온도별(4°C/25°C/37°C/60°C)로 측정 | 진로: 생화학·의생명공학
   - 실험 데이터: 4온도 조건 × 5분 산소 발생량(mL)

4. **식물 호흡과 발효 — 효모의 CO₂ 발생 조건** — 설탕 농도·온도별로 효모 발효 CO₂ 발생량 측정, 최적 발효 조건 도출 | 진로: 식품생명공학·바이오화학
   - 탐구 방법: BTB 색 변화 측정 또는 기체 포집

5. **항생제 내성 — 디스크 확산법으로 보는 세균 감수성** — (학교 허가 균주) 항생제 종류별 억제대 크기(mm) 측정, 내성 문제와 연계 | 진로: 의학·제약학·감염병학
   - 실험 데이터: 항생제 3종 × 3반복, 억제대 직경(mm)

**2학기 — 응용·융합 탐구 (생태계, 진화, 바이오테크)**

6. **CRISPR-Cas9 원리 분석과 윤리적 쟁점** — 유전자 가위의 작동 원리(DNA 이중나선 절단·수선)를 도식화하고, 디자이너 베이비·농업 GMO에 대한 윤리 토론 포함 | 진로: 생명공학·생명윤리학
   - 탐구 방법: 문헌조사 + 원리 다이어그램 + 찬반 논증

7. **미세플라스틱이 식물 생장에 미치는 영향 실험** — 토양 내 미세플라스틱 농도(0/0.1%/0.5%)별 콩나물 생장 높이·무게 측정 | 진로: 환경생명과학·생태학
   - 실험 데이터: 3조건 × 10株 × 14일, 일일 신장 기록

8. **암 세포와 정상 세포의 세포 분열 비교 — 세포주기 이상** — 암 발생 메커니즘(세포주기 체크포인트 붕괴)을 문헌 기반으로 탐구하고, 항암 치료 원리 연계 | 진로: 의학·종양학·생명공학

9. **장내 미생물과 인체 건강 — 프로바이오틱스의 과학** — 발효 식품(요거트·김치) 속 유산균 수를 배지 배양으로 정량화, 가열 처리 시 생존율 비교 | 진로: 식품영양학·의과학
   - 실험 데이터: 처리 조건 2종(생/가열) × 3반복, 콜로니 수(CFU/mL)

10. **생태계 탄소 순환과 기후변화 — 광합성·호흡의 정량 분석** — 식물 종류별 광합성 속도(기포 수/분)를 조도 조건별로 측정하고 탄소 중립 시나리오에 연계 | 진로: 환경생태학·기후과학

### 생활과윤리 (고1)

**1학기 — 기초 탐구 (윤리 이론, 생명·자연 윤리)**

1. **AI 판사가 내린 판결은 정의로운가 — 알고리즘 편향과 공정성** — 미국 COMPAS 시스템 사례로 알고리즘이 인종·성별 편향을 어떻게 재생산하는지 분석하고, 공정성 기준을 윤리 이론으로 평가 | 진로: 법학·AI윤리·사회학
   - 탐구 방법: 문헌조사 + 사례 분석 + 공리주의/의무론 적용
   - 임팩트 이유: 첨단 기술과 윤리 이론을 접목해 실제 사회 문제를 다루는 깊이를 보여줌

   - 탐구 방법: 법률 비교 분석 + 윤리 이론 적용

3. **채식주의는 의무인가 선택인가 — 동물권 윤리 논쟁** — 피터 싱어의 동물 해방론과 임마누엘 칸트의 의무론으로 채식주의 윤리성을 다각도 분석 | 진로: 동물윤리학·법학·철학
   - 탐구 방법: 철학적 논증 분석 + 반론 재구성

4. **기후 정의 — 탄소 배출의 책임은 누구에게 있나** — 선진국·개발도상국 간 탄소 배출 책임 분배를 공리주의·분배적 정의론으로 비교 분석 | 진로: 국제법·환경정책·윤리학
   - 탐구 방법: 통계 데이터(1인당 누적 배출량) + 윤리 이론 적용

5. **뇌사와 식물인간 — 생명의 정의와 존엄사 논쟁** — 뇌사와 식물인간 상태의 의학적·법적·윤리적 차이를 분석하고, 한국·유럽·미국의 존엄사 법제 비교 | 진로: 의학·생명윤리학·법학

**2학기 — 응용·융합 탐구 (사회·직업·정보 윤리, 통일 및 평화)**

## 6. [고1 주제 선정 핵심 전략]

### 고1 특징과 공략 포인트

1. **첫 세특이라 차별화가 쉽다** — 고1은 경쟁자들도 탐구 경험이 없어, 방법론의 체계성(실험 설계·오차 분석)만 보여줘도 돋보임
2. **기초 개념을 현실·트렌드로 연결** — "이차방정식 풀었음"이 아니라 "이차함수로 드론 포물선 경로를 설계"로 서술해야 입팩트
3. **1학기-2학기 연속성** — 1학기 기초 주제와 2학기 융합 주제를 같은 과목에서 이어가면 "성장 스토리"가 만들어짐

### 가장 임팩트 있는 추천 주제 TOP 3 (과목 불문)

| 순위 | 주제 | 이유 |
|------|------|------|
| 1위 | **물리 — 스마트폰으로 측정하는 중력가속도(g)** | 디지털 실험 + 오차 분석 능력 + 일상 도구 활용 창의성이 한 주제에 담김 |
| 2위 | **생윤 — AI 판사의 알고리즘 편향 분석** | 첨단 기술 + 윤리 이론 접목 + 사회 문제 인식, 문과 최강 임팩트 |
| 3위 | **수학 — 기후 데이터 선형 회귀 분석** | 데이터 분석 역량 + 환경 문제 연계 + 이공계/문과 모두 활용 가능 |

### 계열별 추천 전략

**이과 지향 (의학·공학·AI·환경공학)**
- 과학 3과목 실험 중심: 오차 분석·통계 처리 능력 어필
- 수학 2학기 로그함수·회귀분석 주제로 데이터 역량 강조
- 추천 조합: 물리(중력가속도 측정) + 화학(아스피린 합성) + 수학(AI 손실함수)

**문과 지향 (법학·사회학·경제학·국제관계)**
- 생윤 2학기 주제 2~3개로 비판적 분석 역량 집중 어필
- 수학 2학기 확률·통계 주제로 정량 분석 능력 보완
- 추천 조합: 생윤(AI 판사 편향) + 생윤(플랫폼 노동자 권리) + 수학(지니계수)

**예체능 지향 (디자인·음악·스포츠과학)**
- 수학(황금비·화면비율)으로 심미적 수학 감각 어필
- 물리(소리 파동·도플러) → 음향·음악 연계 진로 설명
- 생윤(AI 창작물 저작권)으로 창작자 정체성 + 사회 인식 어필
