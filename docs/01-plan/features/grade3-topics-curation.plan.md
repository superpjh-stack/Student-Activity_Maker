## 1. Overview

### 1.1 Purpose

현재 서비스의 주제 풀은 고1~고3 전체를 커버하는 범용 주제로 구성되어 있다.
고3 학생은 원서 제출(6월 수시)과 수능 후 세특 마무리라는 극도로 현실적인 타임라인 속에 있으며,
"입학사정관에게 임팩트를 주는 주제"가 필요하다.

본 기획은 고3 전용 탐구 주제 50개를 큐레이션하고, 이를 서비스 내 "고3 모드" 또는 별도 주제 풀로 제공하는 것을 목표로 한다.

### 1.2 Background

- 현재 주제 풀: 8개 과목 × 10개 풀, 날짜 시드로 5개 노출 (topics-cache.ts)
- 고3 학생의 페인포인트: "어떤 주제가 의대/공대/이학 학종에서 통하냐"를 모름
- 기회: 최상위권 차별화 주제 + 1학기/2학기 전략을 패키지로 제공하면 전환율 급상승

## 3. 고3 전용 주제 콘텐츠 (50개)

> 기획 원칙:
> - 1학기: 6월 이전 완성 가능, 원서에 직접 기재, 임팩트 극대화
> - 2학기: 수능 후 마무리 or 심층면접 연계, 확장성 강조
> - 모든 주제: 최신 연구 트렌드(2024~2026), 복합 탐구 방법 포함

### 3.1 수학 (고3) — 10개

**1학기 (원서 대비 핵심 탐구)**

1. **베이즈 정리와 의료 진단 정확도 분석** — 조건부 확률을 실제 암 검진 위양성 데이터에 적용해 진단 시스템의 수학적 한계를 분석 | 진로: 의대·보건의료정보학 | 차별화: 실제 NHIS 공개 데이터셋 활용 + 오류율 정량화

2. **미분방정식으로 모델링한 전염병 SIR 모델 검증** — 코로나19 실제 확진 데이터와 SIR 모델 예측값의 오차 분석 및 매개변수 최적화 | 진로: 수리생물학·통계학·공중보건 | 차별화: Python scipy로 수치해 도출 후 실측값 대비 RMSE 계산

3. **프랙탈 차원과 자연 현상의 자기유사성 탐구** — 해안선·나뭇가지·혈관 분기의 하우스도르프 차원을 측정하고 생물학적 최적화 원리와 연결 | 진로: 수학과·물리학과·생명과학 | 차별화: ImageJ 소프트웨어로 실제 이미지 차원 계산

4. **암호학의 수학 — RSA 알고리즘과 소수 분포의 관계** — 리만 가설과 소수 정리를 바탕으로 RSA 키 안전성을 수학적으로 분석하고 양자 컴퓨팅 위협 탐구 | 진로: 컴퓨터공학·수학과·사이버보안 | 차별화: 직접 파이썬으로 소인수분해 시간 측정 실험

5. **게임이론과 내쉬 균형 — 실제 경쟁 상황에 적용** — 죄수의 딜레마부터 경매 이론까지 내쉬 균형 개념을 실제 사례(전기차 시장 경쟁)에 적용해 분석 | 진로: 경제학·경영학·수학과 | 차별화: 보수 행렬 직접 설계 + 균형점 수치 계산

**2학기 (심층면접 연계 탐구)**

6. **위상수학 입문 — 뫼비우스 띠에서 클라인 병까지** — 오일러 지표, 연결성 개념을 활용해 위상 불변량의 의미를 탐구하고 면접에서 "수학의 심층성"을 보여주는 주제 | 진로: 수학과·물리학과 | 차별화: 실제 종이 모형 제작 + 오일러 공식 증명 과정 기록

7. **몬테카를로 시뮬레이션으로 원주율 추정과 복잡계 해석** — 확률적 수치 방법으로 π를 추정하고, 금융·물리 시뮬레이션으로의 확장 원리 탐구 | 진로: 통계학·계량금융·물리학 | 차별화: 시행 횟수 증가에 따른 수렴 속도 그래프 분석

8. **최적화 이론 — 경사하강법과 머신러닝의 수학적 기반** — 다변수 함수의 극값 문제를 신경망 학습 알고리즘과 연결해 AI의 수학적 본질 탐구 | 진로: AI학과·컴퓨터공학·통계학 | 차별화: 직접 2차함수에 경사하강법 구현 후 수렴 조건 분석

9. **수열의 수렴과 카오스 이론 — 로지스틱 맵 탐구** — r값 변화에 따른 로지스틱 맵의 분기 다이어그램을 작성하고 카오스의 수학적 조건 탐구 | 진로: 수학과·물리학과·복잡계과학 | 차별화: 파이썬으로 분기도 직접 생성, 리아푸노프 지수 계산

10. **정수론과 암호화폐 — 타원곡선 암호 ECC 원리 분석** — 타원곡선의 군 구조와 이산로그 문제를 이해하고, 비트코인 서명 알고리즘 ECDSA의 수학적 기초 탐구 | 진로: 수학과·컴퓨터공학·핀테크 | 차별화: 소규모 타원곡선에 직접 점 덧셈 연산 시뮬레이션

### 3.2 물리 (고3) — 10개

**1학기 (원서 대비 핵심 탐구)**

1. **초전도체 마이스너 효과와 양자 자속 양자화** — YBCO 고온 초전도체를 이용한 자기 부상 실험에서 자속 양자화 조건을 검증하고, 2023년 LK-99 논란과 비교 분석 | 진로: 물리학과·재료공학·에너지공학 | 차별화: 실제 액체질소 실험 데이터 + 임계온도 측정

2. **플라즈마 물리와 핵융합 에너지의 가능성** — 토카막 원리와 ITER 프로젝트 현황을 분석하고, 핵융합 점화 조건(로손 기준)을 수식으로 검증 | 진로: 핵공학·물리학과·에너지공학 | 차별화: 2024년 NIF 점화 성공 데이터와 로손 기준 비교 계산

3. **양자 얽힘과 벨 부등식 실험 검증** — 광자 편광 측정 사고 실험을 설계하고, 2022년 노벨 물리학상(Aspect, Clauser, Zeilinger)과 연결해 국소성 한계 분석 | 진로: 물리학과·양자정보과학 | 차별화: 벨 부등식 CHSH 형태를 수학적으로 유도

4. **특수 상대성 이론의 실험적 검증 — GPS 시간 보정** — 로런츠 변환으로 GPS 위성의 시간 팽창량을 계산하고 보정하지 않으면 발생할 위치 오차를 정량화 | 진로: 물리학과·항공우주공학·천문학 | 차별화: 실제 GPS 시간 보정 공식 유도 및 일일 오차(38μs) 계산

5. **포논과 열전 소자 — 제베크 효과의 미시적 원리 탐구** — 온도차로 전기를 생성하는 열전 소자의 제베크 계수를 측정하고, 포논-전자 결합 관점에서 효율 한계를 분석 | 진로: 물리학과·신소재공학·에너지공학 | 차별화: 직접 펠티에 소자로 실험, 온도차-전압 선형성 검증

**2학기 (심층면접 연계 탐구)**

6. **중력파 검출 원리와 LIGO 간섭계 분석** — 마이컬슨 간섭계 원리에서 출발해 LIGO의 10⁻²¹m 정밀도 달성 방법과 2024년 최신 중력파 관측 데이터 탐구 | 진로: 물리학과·천문학·정밀계측공학 | 차별화: 간섭계 공명 주파수 계산 + 민감도 스펙트럼 분석

7. **위상 절연체와 양자 홀 효과** — 물질의 위상 불변량(체른 수) 개념을 이해하고, 2016년 노벨 물리학상과 연결해 미래 양자 소자 응용 분석 | 진로: 물리학과·재료공학·반도체공학 | 차별화: k-공간에서의 베리 위상 개념 도식화

8. **카오스와 나비효과 — 로렌츠 어트랙터 수치 시뮬레이션** — 로렌츠 방정식을 룽게-쿠타법으로 수치 적분하고 초기조건 민감성(리아푸노프 지수)을 정량적으로 분석 | 진로: 물리학과·기상학·복잡계과학 | 차별화: 파이썬으로 3D 어트랙터 직접 시각화

9. **빛의 이중성과 광전 효과의 현대적 응용** — 광전 효과 실험으로 플랑크 상수를 측정하고, 태양전지와 광전자 분광학(XPS)으로 현대 응용 연결 | 진로: 물리학과·광공학·신소재공학 | 차별화: LED 파장별 정지 전압 측정으로 h값 계산

10. **다체계 물리와 창발 현상 — 왜 물은 젖는가** — 물의 표면 장력, 모세관 현상을 통해 미시-거시 연결고리인 창발(emergence) 개념을 탐구하고 생물의학 응용 분석 | 진로: 물리학과·화학과·생명과학부 | 차별화: 다양한 액체로 표면 장력 측정 + 코사인 법칙 검증

### 3.3 화학 (고3) — 10개

**1학기 (원서 대비 핵심 탐구)**

1. **리튬이온 배터리의 전기화학 — 충방전 사이클 열화 분석** — 전기화학 임피던스 분광법(EIS) 개념으로 배터리 내부 저항 변화를 분석하고, 2024년 전고체 배터리 상용화 현황과 비교 | 진로: 화학공학·재료공학·에너지공학 | 차별화: 소형 리튬셀 직접 충방전 사이클 실험, 용량 감소 곡선 측정

2. **약물 키랄성과 탈리도마이드 비극의 화학적 원인** — R/S 배열과 생체 수용체 선택성을 분석하고, FDA의 키랄 약물 승인 기준 변화 탐구 | 진로: 약학과·의대·화학과 | 차별화: 편광계를 이용한 광학 이성질체 회전각 실험

3. **촉매와 효소 — 전이 상태 안정화의 활성화 에너지 측정** — 아레니우스 방정식으로 H₂O₂ 분해 반응에서 MnO₂ 촉매 vs 카탈레이즈 효소의 Ea를 비교 측정 | 진로: 화학과·생화학·약학 | 차별화: 온도별 반응속도 데이터로 아레니우스 플롯 직접 작성

4. **탄소 포집 기술의 화학 — 아민 흡수제와 MOF 비교** — CO₂ 포집 메커니즘을 아민 용액과 금속-유기 골격체(MOF) 관점에서 분석하고, 2025년 CCUS 상용화 현황 평가 | 진로: 화학공학·환경공학·에너지공학 | 차별화: NaOH 용액으로 CO₂ 흡수 실험, 흡수량 적정으로 정량화

5. **계면활성제의 임계 마이셀 농도(CMC) 측정과 세정 메커니즘** — 표면 장력 측정법으로 CMC를 결정하고, 마이셀 내부 소수성 환경이 오염 제거에 활용되는 원리 탐구 | 진로: 화학과·화장품공학·제약공학 | 차별화: 다양한 계면활성제로 CMC 비교, 온도 의존성 분석

**2학기 (심층면접 연계 탐구)**

6. **CRISPR-Cas9의 화학 — DNA 절단 메커니즘의 원자 수준 분석** — Cas9 단백질과 sgRNA의 상호작용, HNH/RuvC 도메인의 핵산분해 화학 메커니즘 탐구 | 진로: 생화학·분자생물학·의대 | 차별화: 모델 구조 분석 + 2020년 노벨 화학상 연결

7. **전기화학적 질소 고정 — 하버-보슈 대안 탐구** — 현재 하버-보슈법의 에너지 문제점 분석 후, 상온 상압 전기화학적 N₂ 환원 반응의 패러데이 효율 측정 개념 탐구 | 진로: 화학공학·환경화학·농학 | 차별화: 질소 환원 반응 전위 계산 + 최신 촉매(Fe, Mo) 논문 분석

8. **고분자 화학과 생분해 플라스틱 — PLA 합성과 분해 속도** — 락타이드 개환 중합으로 PLA 합성 과정을 이해하고, 조건별(온도·pH·미생물) 분해 속도를 측정해 현실성 평가 | 진로: 고분자공학·환경공학·화학과 | 차별화: pH별 PLA 필름 분해 실험, 질량 감소율 측정

9. **반도체 도핑 화학 — n형/p형 불순물 전기전도도 변화** — 실리콘에 인(n형)·붕소(p형) 도핑 효과를 밴드 이론으로 설명하고, 전도도 실험으로 캐리어 농도 추정 | 진로: 반도체공학·재료공학·전자공학 | 차별화: 소금물 농도 변화로 이온 전도도 측정 후 반도체 유추

10. **합성 의약품 분석 — HPLC로 아스피린 순도 측정** — 역상 HPLC 원리를 이해하고, 시판 아스피린의 순도와 불순물 피크를 분석해 의약품 품질 관리 기준 탐구 | 진로: 약학과·분석화학·의대 | 차별화: 표준곡선 작성 + 검출한계(LOD) 계산 과정 포함

### 3.4 생물 (고3) — 10개

**1학기 (원서 대비 핵심 탐구)**

1. **후성유전학 — DNA 메틸화와 유전자 발현 조절의 환경 의존성** — 일란성 쌍둥이의 후성유전체 차이를 사례로 분석하고, 스트레스·영양·오염 노출이 메틸화 패턴에 미치는 영향 탐구 | 진로: 의대·생명과학·유전공학 | 차별화: 시금치 엽록소 추출 + 광도계로 메틸화 간접 측정 실험

2. **장내 마이크로바이옴과 뇌-장 축(Gut-Brain Axis) 탐구** — 장내 세균총이 신경전달물질(세로토닌, GABA) 합성에 관여하는 메커니즘과 2024년 마이크로바이옴 우울증 연구 현황 분석 | 진로: 의대·약학·생명과학 | 차별화: 시판 프로바이오틱스별 균주 성분 비교 분석

3. **CRISPR 유전자 가위와 유전병 치료 — 겸상적혈구 빈혈증 사례** — BCL11A 유전자 편집으로 태아 헤모글로빈 발현을 복구하는 카스게비(Casgevy) 치료법 원리와 2023년 FDA 승인 의의 탐구 | 진로: 의대·분자생물학·생명공학 | 차별화: 혈색소 모형 제작 + HBB 유전자 돌연변이 유형 비교

4. **텔로미어 단축과 세포 노화 — 텔로머레이즈 활성의 역할** — 정상세포 vs 암세포에서 텔로머레이즈 활성 차이를 분석하고, 항노화 연구 최신 동향(2024~2026) 탐구 | 진로: 의대·생화학·노인의학 | 차별화: 양파 뿌리 세포 분열 관찰 + 세포 분열 횟수와 텔로미어 관계 모델링

5. **광합성 효율 최적화 — C3/C4/CAM 식물의 탄소 고정 전략 비교** — 옥수수(C4)와 시금치(C3)의 광합성 속도를 조건별로 비교 측정하고, 기후변화 시대 C4 작물 확대 전략 탐구 | 진로: 농생명공학·식물생물학·환경과학 | 차별화: 엽록체 현미경 관찰 + 광포화점 측정 비교

**2학기 (심층면접 연계 탐구)**

6. **단일세포 RNA 시퀀싱(scRNA-seq) — 세포 다양성 발견의 혁명** — 전통적 벌크 RNA 분석의 한계와 scRNA-seq이 밝혀낸 세포 하위 유형 다양성, 2024년 세포 아틀라스 프로젝트 성과 분석 | 진로: 의대·생명과학·바이오인포매틱스 | 차별화: t-SNE/UMAP 시각화 개념 이해 + 공개 데이터셋 분석

7. **암 면역 치료 — CAR-T 세포의 설계 원리와 한계** — T세포 수용체에 CAR(키메라 항원 수용체)를 삽입하는 유전공학적 과정과 고형암에서의 한계, 2025년 차세대 CAR-NK 동향 탐구 | 진로: 의대·면역학·생명공학 | 차별화: 항원-항체 모형 직접 제작 + 면역 반응 단계 도식화

8. **뉴로플라스티시티 — 학습이 뇌를 바꾸는 신경과학적 메커니즘** — 헵 시냅스 가소성(LTP/LTD)에서 BDNF 발현까지, 학습과 기억의 분자 수준 메커니즘과 신경재생 치료 응용 탐구 | 진로: 의대·신경과학·심리학 | 차별화: 반복 학습 전후 기억 과제 성능 측정 + 수면 효과 실험

9. **진화 게임이론과 이타적 행동의 진화 — 혈연 선택과 상호 이타주의** — 해밀턴 규칙(rb > c)과 팃포탯 전략으로 협력 행동의 진화 메커니즘을 분석하고 사회성 동물 사례에 적용 | 진로: 진화생물학·생태학·심리학 | 차별화: 죄수의 딜레마 시뮬레이션으로 전략별 생존율 계산

10. **합성생물학 — 미니멀 세포(JCVI-syn3.0)와 생명의 최소 조건** — 크레이그 벤터의 최소 유전체 세포 설계 프로젝트를 분석하고, 53개 필수 유전자 기능 불명 문제와 생명 정의 철학적 탐구 | 진로: 생명공학·의대·철학과 | 차별화: 염기서열 데이터베이스에서 syn3.0 유전자 기능 분류 시도

### 3.5 생활과윤리 (고3) — 10개

**1학기 (원서 대비 핵심 탐구)**

1. **AI 도덕 행위자성 논쟁 — ChatGPT는 도덕적 책임 주체가 될 수 있는가** — 행위자성의 철학적 조건(의도·자율성·책임)을 분석하고, 2025년 EU AI Act에서 채택한 법적 접근과 윤리적 함의 비교 | 진로: 철학·법학·AI윤리 | 차별화: 직접 AI와의 대화 실험으로 자율성 여부 귀납적 분석

2. **유전자 편집과 디자이너 베이비 — 생명 윤리의 경계 설정** — 허젠쿠이 사건(2018)부터 2024년 생식세포 편집 모라토리엄 논의까지, 공리주의·의무론·덕 윤리학적 관점 비교 분석 | 진로: 의대·철학·생명윤리 | 차별화: 3가지 윤리 이론 체계적 적용 + 국가별 규제 비교 표

3. **기후 정의 — 탄소 배출 책임의 세대 간·국가 간 분배** — 롤스의 무지의 베일과 공리주의로 탄소 예산 분배 문제를 분석하고, 파리협정과 손실 및 피해(L&D) 기금의 윤리적 타당성 평가 | 진로: 환경정책·철학·법학·국제관계 | 차별화: 국가별 1인당 누적 배출량 데이터로 책임 불균형 정량화

4. **안락사와 존엄사 — 자율성 원리와 생명 보호의 충돌** — 네덜란드·벨기에·캐나다의 MAiD(의료지원사망) 법제와 한국 연명의료결정법을 비교하고, 경사면 논증의 타당성 검토 | 진로: 의대·법학·철학 | 차별화: 실제 사례(고통 기준·동의 요건) 비교 + 생명윤리 원칙 4요소 적용

5. **소셜미디어 알고리즘과 인식론적 거품 — 필터버블의 민주주의 위협** — 에코챔버와 필터버블 개념을 분석하고, 플랫폼의 공론장 의무를 하버마스의 의사소통 합리성으로 평가 | 진로: 사회학·언론정보학·철학 | 차별화: 직접 다른 성향 계정 생성 후 알고리즘 추천 차이 비교 실험

**2학기 (심층면접 연계 탐구)**

6. **데이터 주권과 개인정보 — 프라이버시 권리의 재구성** — GDPR의 잊혀질 권리, 우리나라 개인정보보호법 개정(2024)을 비교하고, 데이터 수집과 개인 자율성 간 균형을 자유주의 관점에서 분석 | 진로: 법학·IT정책·철학 | 차별화: 실제 앱 개인정보 수집 항목 직접 조사 + 필요성 비례성 평가

7. **뇌과학과 자유의지 — 리벳 실험이 도덕 책임에 미치는 함의** — 리벳의 준비 전위 실험 결과를 분석하고, 상용주의(Compatibilism)적 관점에서 신경결정론이 법적·도덕적 책임에 미치는 영향 탐구 | 진로: 철학·신경과학·법학 | 차별화: 실험 재현 시도(반응 시간 측정) + 결정론 반론 논증 구성

8. **동물 실험 윤리와 3R 원칙의 한계** — 대체(Replace)·감소(Reduce)·개선(Refine) 원칙의 실제 적용 한계를 분석하고, 오가노이드·AI 시뮬레이션 대체 기술의 윤리적 의의 평가 | 진로: 수의학·약학·생명윤리 | 차별화: 국내 동물 실험 승인 건수 추이 데이터 분석

9. **공정 무역과 글로벌 공정성 — 개발 원조의 윤리적 역설** — 피터 싱어의 세계 빈곤 논문과 무우야 노코의 원조 비판을 비교하고, 공정 무역 인증 제품의 실제 농가 소득 개선 효과를 데이터로 검증 | 진로: 국제개발·경제학·철학 | 차별화: 공정무역 vs 일반 커피 농가 소득 실증 데이터 비교

10. **트랜스휴머니즘 — 인간 향상 기술의 윤리적 경계** — 뇌-컴퓨터 인터페이스(Neuralink), 유전자 향상, 기억 향상 약물을 인간 존엄성·평등 관점에서 분석하고 2025년 최신 임상 사례 검토 | 진로: 의대·철학·생명공학 | 차별화: 치료 vs 향상의 경계 기준 직접 설정 + 경쟁적 불평등 시나리오 분석

## 5. 입시 전략 분석 콘텐츠

### 5.1 고3만의 특수 전략

- **타임라인 역산 설계**: 6월 수시 원서 마감 기준, 탐구 완성은 5월 말 필수. 4월 이전 주제 확정·실험 데이터 수집 완료가 핵심.
- **학생부 기재 가능성 우선**: 탐구보고서 전체를 학생부에 넣을 수는 없으나, 핵심 내용 3~4문장이 세특에 녹아야 함. AI 생성 시 "세특 압축 버전" 동시 생성 권장.

- **과목 융합 탐구**: 단일 과목보다 수학+물리, 화학+생물 융합 주제가 상위권 대학에서 선호됨. 배치 생성 기능 활용 권장.

| 5위 | 리튬이온 배터리 열화 분석 | 화학 1학기 | 산업 직결, 전기화학 심화, 공학계열 전반 어필 |

### 5.3 학과별 추천 탐구 로드맵

| 진로 계열 | 추천 1학기 주제 | 추천 2학기 주제 | 핵심 차별화 |
|-----------|---------------|---------------|------------|
| 의대/의학계열 | 베이즈+의료진단(수학) + CRISPR+겸상적혈구(생물) | 텔로미어+노화(생물) + 안락사 윤리(생윤) | 정량 데이터 + 임상 연결 + 생명 윤리 |
| 공대 (전기·전자·반도체) | 리튬이온 배터리(화학) + 특수상대론+GPS(물리) | 반도체 도핑(화학) + 위상 절연체(물리) | 실험 데이터 + 공학 응용 + 최신 소자 |
| 이학 (수학·물리·화학) | 게임이론+내쉬균형(수학) + 플라즈마(물리) | 로렌츠 어트랙터(물리) + 고분자+PLA(화학) | 수식 유도 + 시뮬레이션 + 이론 심화 |
| 사범·교육학 | 필터버블+민주주의(생윤) + 광합성 효율(생물) | 뉴로플라스티시티(생물) + 공정무역(생윤) | 교육 적용 관점 + 사회 맥락화 |
| 사회과학·인문·법학 | 기후 정의+탄소분배(생윤) + AI 행위자성(생윤) | 데이터 주권(생윤) + 자유의지(생윤) | 논증 구조 + 실증 데이터 + 국제 비교 |
| 생명공학·약학 | 약물 키랄성+탈리도마이드(화학) + 마이크로바이옴(생물) | 합성생물학+최소세포(생물) + 동물실험 윤리(생윤) | 분자 메커니즘 + 최신 기술 + 윤리 연계 |

### 5.4 1학기 vs 2학기 탐구 완성 전략

| 구분 | 1학기 전략 | 2학기 전략 |
|------|-----------|-----------|
| 목적 | 수시 원서 기재용 최대 임팩트 | 심층면접 준비 + 최종 학생부 기록 |
| 주제 선정 기준 | 실험 데이터 수집 가능, 6월 완성 | 1학기 탐구의 확장·심화, 면접 연결 |
| 탐구 방법 | 실험+통계+문헌고찰 3종 조합 | 이론 심화+현대 연구 연결+사회적 함의 |
| 세특 작성 | 구체적 수치·방법론 중심 | 성장 서사·진로 변화 중심 |
| AI 활용 팁 | 탐구보고서 + 세특 동시 생성 | 면접 예상 질문 + 답변 구조 생성 |
